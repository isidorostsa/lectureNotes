
## 3.6 
$$\min_{\begin{align*}
xy^{2}z^{3}=39\\
x,y,z > 0
\end{align*}}(x^{3} + y^{2}+ z)$$
- $\prod{x_{i}^{\delta_{i}}}\le \sum\limits \delta_{i} x_{i}$
- For $\delta_{i}: \sum\limits \delta_{i}= 1$ this gives: $$\begin{align*}
\delta_{1}x^{3}+ \delta_{2}y^{2}+ \delta_{3}z &\ge x^{3\delta_1}y^{2\delta_2}z^{\delta_3}
\end{align*}$$
We want the experssion on the L.H.S. to match $xy^2z^3$. So we set $$\begin{align*}
\delta_{1}&= \frac{1}{3}\\
\delta_{2}&= 1\\
\delta_{3}&= 3
\end{align*}$$
