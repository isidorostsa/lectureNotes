We represent choice behavior by a *choice structure* of the form:$$(\mathbb{B}, C(\cdot))$$ where:
- $\mathbb{B}$ is a set of nonempty subsets of $X$: $$\forall B \in \mathbb{B}: B \subset X$$ we call elements $B$ *budget sets*, where $\mathbb{B}$ represents all the possible choice combinations.
- $C(\cdot)$ is a rule assigning a nonempty subset $C(B)\subset B$ to every set of elements $B\in\mathbb{B}$ 

examples:

Suppose $X = \{x,y,z\}$ and $\mathbb{B}=\{\{x,y\},\{x,y,z\}\}$. We can have a *choice structure* $(\mathbb{B}, C_1(\cdot))$ where the *choice rule* $C_1$ is: $C_1(\{x,y\})=\{x\}$ and $C_1(\{x,y,z\})=\{x\}$. Here $x$ is always chosen. Another choice structure is $(\mathbb{B}, C_2(\cdot))$ where $C_2(\{x,y\})=\{x\}$ and $C_2(\{x,y,z\})=\{x,y\}$ so whenever the decision maker faces budget $\{x,y,z\}$ she can pick either $x$ or $y$.

We usually restrict the choice structures with the [[weak axiom of revealed preference]]
